# تفقيط الأرقام إلى حروف عربية

استخدم هذه المهارة عند طلب تحويل رقم إلى كلمات عربية.

## الخطوات
1. حدّد الجزء الصحيح والكسور (الهللات/القروش إن وُجدت).
2. فكّك الجزء الصحيح إلى مراتب: آحاد، عشرات، مئات، آلاف، ملايين.
3. اكتب كل مرتبة بالصيغة الصحيحة مع مراعاة:
   - التذكير والتأنيث.
   - العطف بـ«و» بين المراتب (ألف ومئتان وخمسون).
   - المثنى (مئتان، ألفان) والجمع (ثلاثة آلاف).
4. أضف اسم العملة بعد المبلغ إن ذُكرت (ريال، درهم، دينار…).
5. في السياق المالي الرسمي اختم بـ«فقط لا غير».

## أمثلة
- `1250 ريالاً` → «ألف ومئتان وخمسون ريالاً فقط لا غير».
- `305` → «ثلاثمئة وخمسة».
- `2000` → «ألفان».

## ملاحظات
- إن احتمل الرقم صيغتين فاختر الأنسب للسياق المالي.
- هذه مهمة لغوية يؤديها النموذج مباشرةً بلا أدوات تنفيذ.
